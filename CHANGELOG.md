# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-30

### Added
- Initial release
- Pandoc-based converter for high-quality conversion
- Pure Python converter for dependency-free operation
- Automatic image extraction from .docx files
- Auto-detection of .docx files in current directory
- Universal wrapper script for automatic method selection
- Support for headers (H1-H6)
- Support for bold and italic text
- Table conversion
- List conversion (ordered and unordered)
- Comprehensive documentation (5 guides)
- Cross-platform support (macOS, Linux, Windows)
- MIT License

### Features
- `convert.sh` - Universal wrapper script
- `convert_docx_to_md.py` - Pandoc-based converter
- `convert_docx_to_md_python.py` - Pure Python converter
- Automatic output file naming based on input
- Organized image storage in separate folders
- Clear error messages and installation instructions

- README_CONVERTER.md - Technical documentation

- CONTRIBUTING.md - Contribution guidelines
- CHANGELOG.md - This file
